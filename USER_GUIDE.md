# M5StickS3 智能水平仪固件使用说明

## 1. 这个固件是做什么的

这是一个运行在 M5StickS3 上的智能水平仪和测距固件。它使用机身内置 IMU 传感器读取倾斜角度，并通过屏幕和蜂鸣声帮助你判断一个平面是否水平，或者是否达到你设定的目标角度。升级版还支持外接 VL53L0X ToF 模块进行短距离测距。

它适合用来：

- 调整桌面、支架、设备外壳是否水平。
- 调整一个斜面是否回到某个参考角度。
- 粗略测量前后、左右两个方向的倾斜。
- 用蜂鸣声辅助找平，不必一直盯着屏幕。
- 外接 ToF 模块后测量物体到传感器的距离。

## 2. 开机后的界面

烧录完成后，M5StickS3 会先显示：

```text
StickS3 Level
```

然后进入水平仪主界面。

屏幕上主要有三种显示模式：

- `LEVEL`：气泡水平仪模式。
- `ANGLE`：数字角度模式。
- `TARGET`：目标角度模式。
- `DISTANCE`：外接 ToF 测距模式。

短按 `B` 键可以在这三种模式之间切换。

## 3. A 键和 B 键在哪里

在 M5StickS3 上：

- `A` 键：正面屏幕下面的主要按键。
- `B` 键：机身侧边的小按键。

可以简单记成：

```text
正面大按钮 = A
侧边小按钮 = B
```

## 4. 按键操作总表

| 操作 | 功能 |
| --- | --- |
| 短按 `A` | 校准当前姿态为 0 度 |
| 短按 `B` | 切换显示模式 |
| 在 `TARGET` 模式长按 `A` | 目标角度减少 1 度 |
| 在 `TARGET` 模式长按 `B` | 目标角度增加 1 度 |
| 在 `DISTANCE` 模式短按 `A` | 重新检测 ToF 测距模块 |
| 同时长按 `A + B` | 打开或关闭蜂鸣声 |

## 5. P 和 R 分别是什么意思

屏幕里看到的：

```text
P +0.8  R -1.2
```

其中：

- `P` 是 `Pitch`，表示前后方向的俯仰角。
- `R` 是 `Roll`，表示左右方向的横滚角。

简单理解：

- `P` 变化：设备前后方向翘起或压下。
- `R` 变化：设备左右方向倾斜。

如果你短按 `A` 校准，当前姿态会被设为：

```text
P 0.0
R 0.0
```

之后显示的 P 和 R 都是相对于这次校准姿态的偏差。

## 6. 气泡水平仪模式怎么看

`LEVEL` 模式下，屏幕中央有十字和圆圈，小球会随着倾斜方向移动。

目标是：

```text
让小球回到屏幕中心
```

小球颜色含义：

- 绿色：已经很接近水平，误差大约在 1 度以内。
- 黄色：接近水平，但还需要微调。
- 红色：倾斜比较明显。

使用方法：

1. 把 M5StickS3 放在要测量的平面上。
2. 观察小球偏向哪里。
3. 调整平面或设备，让小球逐渐靠近中心。
4. 小球变绿并接近中心时，表示已经基本调平。

注意：这个小球是帮助你找中心的视觉指示。实际使用时不需要纠结正负号，只要让小球回到中心即可。

## 7. 数字角度模式怎么看

`ANGLE` 模式会用大字显示：

```text
P +0.5
R -0.3
```

它适合在你需要读出具体角度时使用。

判断方式：

- P 和 R 都接近 `0.0`：说明接近校准后的水平状态。
- P 偏大：前后方向需要调整。
- R 偏大：左右方向需要调整。

如果屏幕底部显示：

```text
OK
```

表示误差已经在阈值以内。

如果显示：

```text
ADJUST
```

表示还需要继续调整。

## 8. 目标角度模式怎么用

`TARGET` 模式适合用来调整一个指定角度，比如让设备达到 `15°`、`30°` 或某个斜面角度。

界面中会显示：

```text
Target 0 deg
```

表示当前目标 pitch 角度是 0 度。

在 `TARGET` 模式中：

- 长按 `A`：目标角度减少 1 度。
- 长按 `B`：目标角度增加 1 度。

目标角度范围：

```text
-45° 到 +45°
```

屏幕会显示类似：

```text
RIGHT HIGH
FRONT HIGH
ON TARGET
```

含义：

- `RIGHT HIGH`：右侧偏高。
- `LEFT HIGH`：左侧偏高。
- `FRONT HIGH`：前方偏高。
- `BACK HIGH`：后方偏高。
- `ON TARGET`：已经接近目标角度。

## 9. 校准功能怎么用

短按 `A` 会把当前姿态设置为新的 0 度参考。

常见用法有两种。

第一种：测绝对水平。

1. 把设备放到你认为应该水平的地方。
2. 如果你确认这个位置就是参考水平，短按 `A`。
3. 后续小球回到中心就表示回到这个参考状态。

第二种：复制一个斜面角度。

1. 把设备放到某个已知斜面上。
2. 短按 `A` 校准。
3. 再把设备放到另一个平面上。
4. 调整到小球回中心，就表示两个平面的角度接近一致。

## 10. 蜂鸣声怎么看

固件会用蜂鸣辅助找平：

- 接近水平或目标角度时，会发出提示音。
- 倾斜较大时，提示会变少或停止。
- 同时长按 `A + B` 可以静音或恢复蜂鸣。

屏幕右上角会显示：

```text
BEEP
```

或：

```text
MUTE
```

`BEEP` 表示蜂鸣开启，`MUTE` 表示蜂鸣关闭。

## 11. 测距功能怎么用

M5StickS3 本体没有内置测距传感器，所以 `DISTANCE` 模式需要外接 VL53L0X ToF 测距模块。

默认接线：

```text
VL53L0X GND -> StickS3 GND
VL53L0X VIN -> StickS3 5V
VL53L0X SDA -> StickS3 G9
VL53L0X SCL -> StickS3 G10
```

进入 `DISTANCE` 模式后：

- 如果检测到模块，会显示距离，包含 `cm`、`mm` 和 `m`。
- 如果没有检测到模块，会显示 `NO TOF`。
- 如果目标超出有效范围，会显示 `OUT RANGE`。
- 短按 `A` 可以重新检测测距模块。
- 短按 `B` 可以切换回其他模式。

VL53L0X 适合做近距离测量，实际可用距离会受反光率、环境光、目标颜色和模块型号影响。测量黑色、透明或强反光物体时，读数可能不稳定。

## 12. 使用建议

- 第一次使用时，先把设备放在稳定平面上，短按 `A` 校准。
- 测量时尽量让设备底面贴平被测物体。
- 手持测量会有抖动，建议放下后等半秒再读数。
- 如果读数方向和你直觉相反，可以只看小球和提示文字，不必纠结 P/R 正负。
- 如果要更精确，建议多测几次，取稳定后的读数。
- 测距时让 ToF 模块正对目标表面，避免斜着照射。

## 13. 常见问题

### 屏幕上的 P/R 一直跳动

IMU 对轻微震动比较敏感，这是正常现象。把设备放稳，读数会更稳定。

### 小球不在中心，但我觉得设备已经水平

短按 `A` 重新校准当前姿态。如果你需要绝对水平，建议先用一个可靠的水平面作为参考。

### 蜂鸣太吵

同时长按 `A + B` 可以静音。当前版本还没有做音量调节，只支持开关蜂鸣。

### 烧录后没反应

先确认 USB 供电正常，再按一下设备复位。如果仍然没有画面，可以重新执行 PlatformIO 的 `Upload`。

### DISTANCE 页面显示 NO TOF

说明没有检测到 VL53L0X 模块。检查 GND、5V、SDA G9、SCL G10 是否接对，然后在 `DISTANCE` 页面短按 `A` 重试。
